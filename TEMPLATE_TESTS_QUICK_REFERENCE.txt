================================================================================
HOW TO RUN TESTS
================================================================================

Run all marketplace tests:
  cargo test --test template_marketplace_comprehensive
  cargo test --test template_marketplace_workflows

Run specific test:
  cargo test --test template_marketplace_comprehensive test_search_by_tag

Run all tests matching pattern:
  cargo test --test template_marketplace_workflows test_publish

Run with output:
  cargo test --test template_marketplace_comprehensive -- --nocapture

Run single-threaded:
  cargo test --test template_marketplace_comprehensive -- --test-threads=1
